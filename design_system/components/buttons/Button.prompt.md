Pill-shaped primary action button — use for the main CTA on a screen (Start scoring, Add player, Save).

```jsx
<Button variant="primary" icon="plus" onClick={() => {}}>Add player</Button>
```

Variants: `primary` (coral, filled), `secondary` (teal, filled), `ghost` (transparent, text-only), `outline` (bordered). Sizes: `sm`, `md`, `lg`. Press state scales down 0.97 + reduces shadow; hover darkens the fill one step; disabled goes flat neutral with no shadow.
